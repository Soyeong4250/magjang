## Vue 네이버 버튼 활성화

> 스켈레톤을 가지고 수정하기엔 내부 구조를 완벽히 이해하지 못 했으니 비틀거리더라도 스스로 만들어보는 것을 계획으로 세움

- `vue create frontend` 후 frontend 폴더에서 vscode 다시 엶
- `vue add vuex`
- `npm run serve`로 구동 테스트
- `npm install bootstrap`
- `import "bootstrap/dist/css/bootstrap.min.css";`
- 네이버 버튼 컴포넌트 생성 후 추가
