+ 컬렉션+함수형, 데이터처리 연산을 지원하도록 소스에서 추출된 연속적인 요소

+ 주요 패키지, 클래스, 메서드
	> map(), filter(), reduce(), min()...

+ 중간연산은 스트림을 반환, 여러 연산을 조합할 수 있음
